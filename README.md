# Lista de tareas en php
Proyecto: Lista de tareas con php | DATW

### Creación de la base de datos para una alicacion tipo TO DO:

Se tiene que crear una aplicacion del tipo TO DO en la que los usuarios(id,Nombre, apellidos, usuario, contraseña) creen listas de tareas (id,nombre), estas listas tienen tareas(id,nombre,descripcion,fecha, fecha_vencimiento,estado, prioridad)
